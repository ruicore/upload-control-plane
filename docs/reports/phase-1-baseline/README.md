# Phase 1 Baseline Reports

Status: active

This directory records Phase 1 baseline evidence for the Agentic Repository
Execution Standard project.

Phase 1 starts after Phase 0. Phase 0 defined the project charter,
deliverable boundaries, human and agent autonomy boundaries, experiment
design, and document structure. Phase 1 is dedicated to baseline audit
preparation and baseline reporting only.

This directory is intentionally narrow. It may record repository-specific
baseline boundary facts, baseline audit results, hotspot observations,
architecture drift observations, and baseline evidence generated during Phase
1. It must not define reusable execution standards, create `AGENTS.md`, draft
publication material, or modify production implementation.

For Task P1-01, this directory contains only Git boundary documentation:

- `git-boundary.md`: the branch, tag, and HEAD commit boundary at the start of
  Phase 1 baseline work.

Later Phase 1 documents should remain repository-specific reports. Reusable
standards and templates belong in later phases and different directories.
